## ğŸ› ï¸ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/r3dg0d/DarkMAGARebornBot.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure `.env`:
   ```env
   TOKEN=your_discord_token
   CLIENT_ID=your_client_id
   GUILD_ID=your_guild_id
   FAL_KEY=your_fal_ai_key
   OPENAI_API_KEY=your_openai_key
   ```

4. Deploy commands:
   ```bash
   npm run deploy
   ```

5. Start the bot:
   ```bash
   npm start
   ```
